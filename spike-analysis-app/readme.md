## Requires

Python 3.7.x

## Install

    # download and extract .zip or clone with git
    git clone https://github.com/cudmore/bAnalysis.git
    
    cd spike-analysis
    pip install -r requirements.txt

## Run

    python src/AnalysisApp.py

## Building a standalong app (macOS)

Install pyinstaller

    pip install pyinstaller

Make the app

    cd src
    ./makeapp
